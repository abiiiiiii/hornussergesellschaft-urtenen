# Hornussergesellschaft Urtenen

Angular Project for the webpage of the Hornussergesellschaft Urtenen.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Deploy to Firebase

To deploy the webpage to firebase, you have to do the following steps:
- `npm install -g firebase-tools`
- `firebase login` - Login to your Firebase Account
- run `ng build`
- `firebase deploy` - Deploys the generated code to firebase

## Running unit tests

Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).
